# Documentation

Create your documentation

## Tool and Theme

- Mkdocs
- Mkdocs Material

## Quick start

Install [python](https://www.python.org/downloads/)

Install Mkdocs Material theme

```powershell
    pip install mkdocs-material
```

Run the project

```powershell
    mkdocs serve
```
